# Light Speed Logistics

Welcome to the Light Speed Logistics landing page project. This application has been converted to [Next.js](https://nextjs.org/) (App Router).

## Getting Started

### Prerequisites

Ensure you have Node.js & npm installed.

### Installation

Clone the repository and install dependencies:

```bash
git clone <YOUR_GIT_URL>
cd light-speed-landing-main
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app`: Contains the App Router pages and layouts.
- `src/components`: Reusable UI components.
- `src/assets`: Static assets like images and Lottie animations.

## Technologies Used

- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn-ui
- **Animations**: Framer Motion, Lottie Web
